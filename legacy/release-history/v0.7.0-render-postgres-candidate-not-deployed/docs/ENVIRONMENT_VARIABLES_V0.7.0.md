# Environment Variables — v0.7.0

## Required Render API variables

| Variable | Secret | Purpose |
|---|---:|---|
| `DATABASE_URL` | Yes | Injected from Render PostgreSQL internal connection URL |
| `ADMIN_EMAIL` | No/Personal | Business owner login email |
| `ADMIN_PASSWORD` | Yes | At least 12 characters; new owner/recovery password |
| `JWT_SECRET` | Yes | At least 32 random characters; generated by Blueprint by default |
| `ALLOWED_ORIGINS` | No | Exact comma-separated Guide, Chat, and Admin origins |
| `DEEPSEEK_API_KEY` | Yes | DeepSeek API authentication |
| `R2_ACCOUNT_ID` | Yes | Cloudflare account ID |
| `R2_ACCESS_KEY_ID` | Yes | R2 S3 API access key ID |
| `R2_SECRET_ACCESS_KEY` | Yes | R2 S3 API secret |
| `R2_BUCKET_NAME` | No | Production default: `bdg-ai-guide-images` |

## Important optional variables

| Variable | Default | Purpose |
|---|---:|---|
| `RESET_OWNER_PASSWORD_ON_DEPLOY` | `false` | Set true for one deployment to replace imported owner password |
| `DB_POOL_MAX` | `10` | Maximum PostgreSQL application pool size |
| `DB_CONNECT_TIMEOUT_MS` | `5000` | Connection timeout |
| `DB_QUERY_TIMEOUT_MS` | `15000` | Query timeout |
| `DEEPSEEK_TIMEOUT_MS` | `15000` | AI provider timeout |
| `RATE_LIMIT_CHAT` | `30` | Chat requests per IP/window |
| `RATE_LIMIT_LOGIN` | `10` | Login requests per IP/window |
| `MAX_REQUEST_BYTES` | `20971520` | Maximum request body, including uploads |

## Cloudflare Pages build variables

- Guide Pro: `VITE_API_BASE`
- Chat Pro: `VITE_BDG_API_BASE` and `VITE_API_BASE`
- Admin Pro: `VITE_API_BASE_URL`

Use the same HTTPS Render API/custom-domain URL for all three production builds. Do not include a trailing slash.
