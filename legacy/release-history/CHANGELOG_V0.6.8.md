# Changelog — v0.6.8

## Fixed

- Removed public Guide Lovable/demo fallback categories.
- Removed public Guide demo featured cards and stock Unsplash guide fallback image.
- Public Guide now displays backend-only categories, guides and FAQs.
- Empty states now tell Admin to publish real content instead of showing fake examples.
- Guide cards can render without a cover image if backend has none.
- Added deployment scripts for v0.6.8.
- Worker health version updated to `0.6.8-worker`.

## Scope

This release intentionally focuses on public Guide backend binding and demo data removal only. It does not change AI/chat behavior.
